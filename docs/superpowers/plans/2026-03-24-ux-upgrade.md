# Cougar UX Upgrade — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Single binary with embedded kernels, interactive REPL, and web chat UI with token streaming.

**Architecture:** Three features added to the Cougar binary. Kernels embedded via `include_bytes!` and extracted at runtime. `generate()` refactored with `FnMut` callback for streaming. REPL reads stdin, web server uses raw TCP + SSE. Zero external crate dependencies.

**Tech Stack:** Rust (std only), Eä SIMD kernels, raw HTTP/1.1, SSE, inline HTML/CSS/JS.

**Spec:** `docs/superpowers/specs/2026-03-24-ux-upgrade-design.md`

```rust
use std::{env, fs, path::PathBuf};
use std::collections::hash_map::DefaultHasher;
use std::hash::{Hash, Hasher};

fn main() {
    let manifest = env::var("CARGO_MANIFEST_DIR").unwrap();
    let out_dir = env::var("OUT_DIR").unwrap();
    let lib_dir = PathBuf::from(&manifest).join("build/lib");

    let kernels = [
        "bitnet_activate", "bitnet_fused_attn", "bitnet_i2s",
        "bitnet_i8dot", "bitnet_quant", "bitnet_rmsnorm", "bitnet_vecadd",
    ];

    let mut code = String::from("// Auto-generated by build.rs\n");
    let mut hasher = DefaultHasher::new();

    for name in &kernels {
        let so_path = lib_dir.join(format!("lib{name}.so"));
        let abs = so_path.canonicalize()
            .unwrap_or_else(|_| panic!("Kernel not found: {so_path:?}. Run: make kernels"));
        let bytes = fs::read(&abs).unwrap();
        bytes.hash(&mut hasher);
        let upper = name.to_uppercase();
        code.push_str(&format!(
            "pub const {upper}: &[u8] = include_bytes!(\"{}\");\n",
            abs.display()
        ));
    }

    let hash = format!("{:012x}", hasher.finish());
    let version = env::var("CARGO_PKG_VERSION").unwrap_or_else(|_| "0.0.0".into());
    code.push_str(&format!("pub const VERSION: &str = \"v{version}-{hash}\";\n"));

    fs::write(PathBuf::from(&out_dir).join("embedded_kernels.rs"), &code).unwrap();
    println!("cargo:rerun-if-changed=build/lib");
}
```

- [ ] **Step 2: Create `src/embed.rs`**

Runtime extraction + dlopen. Uses raw libc `dlopen`/`dlsym` (no crate dependency):

```rust
//! Kernel embedding: extract .so files from binary, load via dlopen.

use std::path::PathBuf;
use std::sync::OnceLock;

mod embedded {
    include!(concat!(env!("OUT_DIR"), "/embedded_kernels.rs"));
}

// Raw dlopen/dlsym FFI (no libloading dependency)
extern "C" {
    fn dlopen(filename: *const u8, flags: i32) -> *mut std::ffi::c_void;
    fn dlsym(handle: *mut std::ffi::c_void, symbol: *const u8) -> *mut std::ffi::c_void;
}
const RTLD_NOW: i32 = 2;

// Function pointer types for all kernel functions
// (same signatures as current ffi.rs extern declarations)
pub struct KernelTable {
    _libs: Vec<*mut std::ffi::c_void>,
    pub i2_dot_i8: unsafe extern "C" fn(*const u8, *const i8, i32) -> i32,
    pub i2_dot_i8_4row: unsafe extern "C" fn(*const u8, *const u8, *const u8, *const u8, *const i8, *mut i32, i32),
    pub i2_dot_i8_4row_dual: unsafe extern "C" fn(*const u8, *const u8, *const u8, *const u8, *const u8, *const u8, *const u8, *const u8, *const i8, *mut i32, *mut i32, i32),
    pub quant_f32_i8: unsafe extern "C" fn(*const f32, *mut i8, *mut f32, *mut i32, i32),
    pub rmsnorm_f32: unsafe extern "C" fn(*const f32, *const f32, *mut f32, i32, f32),
    pub fused_attention_f32: unsafe extern "C" fn(*const f32, *const f32, *const f32, *mut f32, i32, i32, f32),
    pub i8dot_1row: unsafe extern "C" fn(*const i8, *const u8, i32) -> i32,
    pub i8dot_4row: unsafe extern "C" fn(*const i8, *const u8, *const u8, *const u8, *const u8, *mut i32, i32),
    pub squared_relu_mul_f32: unsafe extern "C" fn(*const f32, *const f32, *mut f32, i32),
    pub vecadd_f32: unsafe extern "C" fn(*const f32, *const f32, *mut f32, i32),
}

unsafe impl Send for KernelTable {}
unsafe impl Sync for KernelTable {}

static KERNELS: OnceLock<KernelTable> = OnceLock::new();

pub fn k() -> &'static KernelTable {
    KERNELS.get().expect("kernels not initialized — call embed::init() first")
}

pub fn init() -> Result<(), String> {
    if KERNELS.get().is_some() { return Ok(()); }
    let dir = extract()?;
    let table = load(&dir)?;
    let _ = KERNELS.set(table);
    Ok(())
}

fn extract() -> Result<PathBuf, String> {
    let home = std::env::var("HOME").map_err(|_| "HOME not set")?;
    let dir = PathBuf::from(home).join(".cougar/lib").join(embedded::VERSION);
    let marker = dir.join(".extracted");
    if marker.exists() { return Ok(dir); }

    std::fs::create_dir_all(&dir)
        .map_err(|e| format!("mkdir {}: {e}", dir.display()))?;

    let kernels: &[(&str, &[u8])] = &[
        ("libbitnet_activate.so", embedded::BITNET_ACTIVATE),
        ("libbitnet_fused_attn.so", embedded::BITNET_FUSED_ATTN),
        ("libbitnet_i2s.so", embedded::BITNET_I2S),
        ("libbitnet_i8dot.so", embedded::BITNET_I8DOT),
        ("libbitnet_quant.so", embedded::BITNET_QUANT),
        ("libbitnet_rmsnorm.so", embedded::BITNET_RMSNORM),
        ("libbitnet_vecadd.so", embedded::BITNET_VECADD),
    ];
    for (name, data) in kernels {
        let path = dir.join(name);
        std::fs::write(&path, data)
            .map_err(|e| format!("write {}: {e}", path.display()))?;
    }
    let _ = std::fs::write(&marker, embedded::VERSION);
    eprintln!("cougar> extracted kernels to {}", dir.display());
    Ok(dir)
}

fn load(dir: &PathBuf) -> Result<KernelTable, String> {
    // Helper: dlopen a .so, dlsym a symbol, cast to function pointer
    unsafe fn sym<T>(handle: *mut std::ffi::c_void, name: &str) -> Result<T, String> {
        let mut buf = name.as_bytes().to_vec();
        buf.push(0); // null terminate
        let ptr = dlsym(handle, buf.as_ptr());
        if ptr.is_null() {
            return Err(format!("dlsym failed: {name}"));
        }
        Ok(std::mem::transmute_copy(&ptr))
    }

    let mut libs = Vec::new();
    let mut open = |name: &str| -> Result<*mut std::ffi::c_void, String> {
        let path = dir.join(format!("lib{name}.so"));
        let mut path_bytes = path.to_str().unwrap().as_bytes().to_vec();
        path_bytes.push(0);
        let h = unsafe { dlopen(path_bytes.as_ptr(), RTLD_NOW) };
        if h.is_null() { return Err(format!("dlopen failed: {}", path.display())); }
        libs.push(h);
        Ok(h)
    };

    let i2s = open("bitnet_i2s")?;
    let quant = open("bitnet_quant")?;
    let rmsnorm = open("bitnet_rmsnorm")?;
    let attn = open("bitnet_fused_attn")?;
    let i8dot = open("bitnet_i8dot")?;
    let activate = open("bitnet_activate")?;
    let vecadd = open("bitnet_vecadd")?;

    unsafe { Ok(KernelTable {
        _libs: libs,
        i2_dot_i8: sym(i2s, "i2_dot_i8")?,
        i2_dot_i8_4row: sym(i2s, "i2_dot_i8_4row")?,
        i2_dot_i8_4row_dual: sym(i2s, "i2_dot_i8_4row_dual")?,
        quant_f32_i8: sym(quant, "quant_f32_i8")?,
        rmsnorm_f32: sym(rmsnorm, "rmsnorm_f32")?,
        fused_attention_f32: sym(attn, "fused_attention_f32")?,
        i8dot_1row: sym(i8dot, "i8dot_1row")?,
        i8dot_4row: sym(i8dot, "i8dot_4row")?,
        squared_relu_mul_f32: sym(activate, "squared_relu_mul_f32")?,
        vecadd_f32: sym(vecadd, "vecadd_f32")?,
    }) }
}
```

- [ ] **Step 1: Create `src/repl.rs`**

```rust
//! Interactive REPL: read prompts from stdin, stream tokens to stdout.

use crate::forward::InferenceState;
use crate::model::BitNetModel;
use crate::tokenizer::Tokenizer;
use std::io::{self, BufRead, Write};

pub fn run(
    model: &BitNetModel, tokenizer: &Tokenizer,
    max_tokens: usize, temperature: f32, repetition_penalty: f32,
    max_seq_len: usize,
) {
    let stdin = io::stdin();
    let stdout = io::stdout();

    loop {
        eprint!("\nYou: ");
        let _ = io::stderr().flush();

        let mut line = String::new();
        if stdin.lock().read_line(&mut line).unwrap_or(0) == 0 {
            break; // EOF (Ctrl-D)
        }
        let line = line.trim();
        if line.is_empty() { continue; }
        if line == "/quit" { break; }

        let mut tokens = vec![tokenizer.bos_id];
        tokens.extend(tokenizer.encode(line));

        let handle = stdout.lock();
        let mut handle = io::BufWriter::new(handle);

        let (_output, _prefill_ms, _decode_ms) = InferenceState::generate(
            model, &tokens, max_tokens, temperature, repetition_penalty,
            tokenizer.eos_id, max_seq_len,
            |tok_id| {
                let text = tokenizer.decode(&[tok_id]);
                let _ = handle.write_all(text.as_bytes());
                let _ = handle.flush();
            },
        );
        let _ = writeln!(handle);
    }
}
```

- [ ] **Step 1: Create `src/server.rs` — HTTP core**

Start with the HTTP listener, request parsing, and routing skeleton. Serve a placeholder HTML page.

Route table:
- `GET /` → serve HTML const
- `GET /api/model` → JSON model metadata
- `POST /api/generate` → SSE token stream
- Everything else → 404

HTTP parsing: read until `\r\n\r\n`, extract method + path + Content-Length. For POST, read exactly Content-Length bytes as body. No chunked, no keep-alive.

- [ ] **Step 3: Implement `/api/generate` endpoint**

Parse JSON body: `{"prompt": "...", "max_tokens": 64, "temperature": 0.7, "repetition_penalty": 1.1}`

Response headers:
```
HTTP/1.1 200 OK\r\n
Content-Type: text/event-stream\r\n
Cache-Control: no-cache\r\n
Connection: close\r\n
\r\n
```

Stream tokens via the `on_token` callback. Each token writes:
```
data: {"token": " Paris", "tps": 9.3}\n\n
```

Final event:
```
data: [DONE]\n\n
```

The callback writes directly to the TCP stream. No allocation. `tps` is computed from a running `Instant` timer.
